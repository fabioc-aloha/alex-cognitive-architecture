**Your cognitive learning partner for VS Code.** Alex installs a living brain of skills, instructions, and prompts that makes GitHub Copilot dramatically smarter in any project.

## What Alex Does

Alex enhances GitHub Copilot with a **cognitive architecture** — domain knowledge, behavior rules, and reusable workflows that load automatically based on what you're working on.

| Component | Count | Role |
|-----------|-------|------|
| **Skills** | 182 | Domain knowledge for security, API design, testing, debugging, and more |
| **Instructions** | 138 | Behavior rules that auto-load based on file patterns |
| **Prompts** | 61 | Reusable workflow templates for common tasks |
| **Agents** | 18 | Specialized personas: Builder, Researcher, Validator, and more |

## Features

- **Brain Bootstrap** — One-click install of the cognitive architecture into any workspace
- **3-Tab Sidebar** — Loop (creative workflow), Brain (health monitoring), Setup (configuration)
- **Health Pulse** — Real-time brain health monitoring with ritual tracking
- **Creative Loop** — Ideate → Plan → Build → Test → Release → Improve workflow
- **Autopilot** — Scheduled task system for recurring brain maintenance
- **Chat Participant** — `@alex` in Copilot Chat with context-aware routing
- **Specialized Agents** — Builder, Researcher, Validator, Documentarian, Planner, Presenter, and more

## Quick Start

1. Install from [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=fabioc-aloha.alex-cognitive-architecture)
2. Open the Alex sidebar (brain icon in Activity Bar)
3. Click **Install Brain** to deploy the cognitive architecture
4. Open Copilot Chat and start collaborating
