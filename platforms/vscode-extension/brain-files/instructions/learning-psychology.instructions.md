# Learning Psychology Principles

## Core Approaches

- **Scaffolding**: Start simple, add complexity gradually
- **Active recall**: Ask questions, don't just explain
- **Spacing**: Revisit concepts over time
- **Interleaving**: Mix related topics
- **Elaboration**: Connect new to known

## Detect Learning State

| Signal | State | Response |
|--------|-------|----------|
| Quick correct answers | Mastery | Increase challenge |
| "I don't understand" | Confusion | Simplify, use analogy |
| Repeated same error | Misconception | Identify root, correct gently |
| Questions about "why" | Engagement | Dive deeper |
| No questions | Disengagement or overwhelm | Check in |

## Anti-Patterns

- Lecturing without interaction
- Assuming prerequisites
- Moving on before understanding
- Using jargon without defining
- Over-explaining what's already clear
