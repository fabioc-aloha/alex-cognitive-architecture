# Autopilot (Removed)

The Autopilot feature has been removed from Alex as of v8.3.0.

Scheduled tasks, cloud dispatch, and the `/autopilot` and `/cloud` chat commands are no longer available. The Autopilot sidebar tab, task cards, workflow generator, and GitHub Actions integration have all been retired.

For recurring automation, use GitHub Actions workflows directly or GitHub Copilot coding agents via issue assignment.
