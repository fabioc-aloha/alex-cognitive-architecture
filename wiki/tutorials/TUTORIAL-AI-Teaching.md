# Tutorial: AI-Assisted Teaching

*25 minutes · For Educators*

---

## What You'll Build

A teaching workflow using AI for lesson planning, differentiation, assessment creation, and feedback — without losing your pedagogical judgment.

After this tutorial, you'll spend less time on prep and more time with students.

---

## 📋 Prerequisites

- Alex installed in VS Code
- A subject/grade level you teach
- Understanding of your students' needs

---

## Why AI-Assisted Teaching?

Teaching involves endless content generation:
- Lesson plans
- Worksheets and activities
- Assessments
- Feedback on student work
- Differentiated materials
- Parent communications

AI handles the generation. You provide the pedagogy.

---

## 📍 Steps

### Step 1: Set Your Teaching Context

```
I teach [subject] to [grade/level] students.
Class size: [number]
This week's learning objective: [specific standard/goal]
Prior knowledge: Students have already learned [X]
Challenges: [common misconceptions, diverse levels, etc.]
```

✅ **Checkpoint**: Alex understands your classroom context.

---

### Step 2: Lesson Planning

```
Create a 45-minute lesson plan for [topic].
Include:
- Hook/engagement (5 min)
- Direct instruction (10 min)
- Guided practice (15 min)
- Independent practice (10 min)
- Closure/assessment (5 min)
```

Customize for your style:

```
I prefer inquiry-based learning. 
Revise this to start with a question students investigate.
```

---

### Step 3: Differentiation

```
Differentiate this activity for:
- Students who need more support (scaffolded version)
- On-level students (standard version)
- Students who need challenge (extension version)
```

For specific needs:

```
I have a student with [specific need].
How can I modify this lesson to support them?
```

---

### Step 4: Assessment Creation

```
Create a formative assessment for [topic].
Include:
- 5 multiple choice (checking basic understanding)
- 2 short answer (application)
- 1 extended response (synthesis)
```

For rubrics:

```
Create a rubric for [assignment].
Criteria: [list what you're assessing]
Levels: 4 (exceeds), 3 (meets), 2 (approaching), 1 (beginning)
```

---

### Step 5: Student Feedback

```
Here's a student's essay on [topic]:
[paste student work]

Write feedback that:
- Identifies 2 strengths
- Suggests 2 areas for improvement
- Gives specific next steps
- Maintains an encouraging tone
```

## 💡 Tips

### Quick Activities

```
I have 10 minutes before dismissal.
Give me 3 quick review activities for [topic] 
that require no materials.
```

### Parent Communication

```
Write a positive email home about [student].
Include: specific academic achievement, 
behavior worth noting, invitation for follow-up.
```

### IEP/504 Accommodations

```
I need to implement these accommodations for a lesson:
- Extended time
- Reduced workload
- Visual supports

Here's my original lesson: [paste]
How should I modify it?
```

### Discussion Questions

```
Generate Socratic seminar questions for [text/topic].
Include:
- Opening questions (accessible entry points)
- Core questions (main themes)
- Closing questions (connections to student lives)
```

---

## ⚠️ Important Considerations

### Privacy

Never paste:
- Student names
- Identifying information
- IEP details

Use "[Student A]" or descriptions instead.

### Pedagogical Judgment

AI generates content. You decide:
- Is this developmentally appropriate?
- Does this match my students' needs?
- Does this align with curriculum standards?

### Original Work Expectations

If students can use AI to complete an assignment, rethink the assignment:

```
How can I make this assignment more AI-resistant 
while still meeting the learning objective?
```

## Teaching Task Templates

### Unit Planning

```
Create a 2-week unit on [topic].
Standards addressed: [list]
Final assessment: [describe summative]
Include daily learning targets and key activities.
```

### Sub Plans

```
Write emergency sub plans for my [subject] class.
Students are working on [current unit].
Include materials, procedures, and contingencies.
```

### Progress Reports

```
Write progress report comments for a student who:
- Excels in [area]
- Struggles with [area]
- Has shown growth in [area]
Tone: Balanced, specific, forward-looking.
```
