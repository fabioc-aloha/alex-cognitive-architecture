# Tutorials

Step-by-step guides for common tasks with Alex.

---

## How Tutorials Work

Each tutorial takes you from start to finish on a specific task. They assume you have Alex installed and working but don't assume expertise in the topic.

**Estimated times** are realistic, not optimistic. Your first run might take longer as you explore.

## Tutorial Conventions

- 📋 **Prerequisites** — What you need before starting
- 📍 **Steps** — Numbered actions to follow
- 💡 **Tips** — Optional enhancements
- ⚠️ **Warnings** — Things that can go wrong
- ✅ **Checkpoints** — How to verify you're on track
