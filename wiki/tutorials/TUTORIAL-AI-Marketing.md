# Tutorial: AI-Assisted Marketing

![AI marketing](images/tutorial-ai-marketing.png)

*25 minutes · For Marketing Professionals*

## What You'll Build

A marketing workflow using AI for campaign briefs, copy generation, brand voice consistency, and multi-format content creation.

After this tutorial, you'll produce more content without sacrificing quality.

---

## 📋 Prerequisites

- Alex installed in VS Code
- A marketing task (campaign, copy, content)
- Your brand guidelines (if available)

---

## Why AI-Assisted Marketing?

AI accelerates:
- First-draft generation
- Multi-format adaptation
- A/B variant creation
- Headline brainstorming
- Consistency checking

AI still needs:
- Brand judgment
- Strategic direction
- Audience intuition
- Final approval

---

## 📍 Steps

### Step 1: Establish Brand Context

Before any content, ground Alex in your brand:

```
Our brand is [company name] in [industry].
Voice: [describe — professional, playful, authoritative, etc.]
Audience: [describe primary customer]
We avoid: [terms, tones, claims to avoid]
Competitors: [list main competitors]
```

✅ **Checkpoint**: Alex understands your brand positioning.

---

### Step 2: Campaign Brief

```
I'm creating a campaign for [product/launch/initiative].
Goal: [awareness, leads, conversions, retention]
Timeline: [dates]
Channels: [email, social, paid, content]
Budget: [if relevant]
Key message: [what we want people to remember]
```

Alex can help structure the brief:

```
Draft a creative brief for this campaign.
Include: objective, audience, key messages, deliverables, success metrics.
```

---

### Step 3: Generate Copy

Work by format:

**Email:**
```
Write 3 subject line options for our product launch email.
Audience: existing customers.
Tone: excited but not hypey.
```

**Social:**
```
Adapt this message for LinkedIn, Twitter, and Instagram.
LinkedIn: professional, longer.
Twitter: punchy, under 280 chars.
Instagram: visual-first, emoji-friendly.
```

**Landing Page:**
```
Write hero copy for our landing page.
Include: headline, subhead, 3 bullet benefits, CTA.
```

---

### Step 4: Create Variants

For A/B testing:

```
Create 3 variants of this headline:
1. Benefit-focused
2. Curiosity-driven
3. Social proof angle
```

For personalization:

```
Adapt this email for:
- Enterprise buyers (focus on scale, security)
- SMB buyers (focus on ease, price)
- Technical evaluators (focus on features, integration)
```

---

### Step 5: Review for Brand Consistency

```
Review this copy against our brand guidelines:
- Does the tone match our voice?
- Are we using approved terminology?
- Any claims that need verification?
```

## 💡 Tips

### Content Repurposing

```
Turn this blog post into:
- 5 LinkedIn posts
- 1 email newsletter section
- 3 tweet threads
- 1 infographic outline
```

### Competitive Analysis

```
How does our messaging compare to [competitor]?
What differentiates our positioning?
```

### SEO Optimization

```
Review this content for SEO:
- Keyword usage
- Header structure
- Meta description
- Internal linking opportunities
```

### Customer Story Framework

```
I have these customer quotes and results:
[paste data]

Draft a case study following:
Challenge → Solution → Results format.
```

---

## ⚠️ Common Mistakes

### Generic Copy

AI defaults to generic when it lacks context. Always provide:
- Specific audience
- Brand voice guidelines
- Competitive positioning

### Over-Promising

AI may generate claims you can't back up. Review for:
- Superlatives ("best," "only")
- Quantitative claims
- Promises that need legal review

### Losing Your Voice

If content sounds "AI-generated," iterate:
```
This sounds generic. Make it more [specific quality].
Use language our audience actually uses.
```

## Marketing Task Templates

### Product Launch

1. Frame: "Launch [product] to [audience] via [channels]"
2. Brief: Generate campaign brief
3. Copy: Create per-channel content
4. Variants: A/B test options
5. Review: Brand consistency check

### Content Calendar

```
Create a month of content ideas for [topic area].
Mix: educational (40%), promotional (30%), engagement (30%).
Formats: blog, social, email.
```

### Campaign Performance Analysis

```
Here are our campaign results:
[paste metrics]

What worked? What should we change?
Draft recommendations for next quarter.
```
